# Technology Stack

## Programming Language
- **TypeScript 5:** Strongly typed superset of JavaScript, offering excellent type safety and tooling.
- **Node.js 24:** Javascript runtime built on Chrome's V8 JavaScript engine.

## Backend Framework
- **Express 5:** Fast, unopinionated, minimalist web framework for Node.js.

## Database & ORM
- **PostgreSQL 16:** Powerful, open-source object-relational database system.
- **Drizzle ORM:** Headless, type-safe ORM for TypeScript and PostgreSQL.

## Caching & Background Jobs
- **Redis:** In-memory data structure store, used as a database, cache, and message broker.
- **BullMQ:** Message queue and background job processing system based on Redis.

## Tooling & Utilities
- **Zod:** TypeScript-first schema declaration and validation library.
- **Inversify:** Powerful and lightweight inversion of control container.
- **OpenAPI / Swagger UI:** Automated API documentation generation.
